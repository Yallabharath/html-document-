ABOUT HTML

HTML is known as"HYPER TEXT MARKUP LANGAUGE".It is used to interact with browser & to create web page.Most of we use HTML-5 version discover by "TIM BERNERS LEE".

-In html language we can write using tag(< open tag)(> closing tag),the content will be written between both tags.

****TYPES OF TAGS****

1.container tags /opening and closing tags

SYNTAX:-< tagname > attribute content < /tag name >

2.self closing tags /empty tags/void elements

SYNTAX:- < /tag name >

< DOCTYPE >:- All HTML documents must start with a declaration. The declaration is not an HTML tag.

< HTML >:-The tag represents the root of an HTML document. The tag is the container for all other HTML elements (except for the tag).

< HEAD >:-The tag is a container for metadata and links to external resources.

< BODY >:-The tag defines the document's body. The element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

~~~~[HEADING TAGS IN HTML]~~~~


Heading tags are used to define headings of documents. You can use different sizes for your headings. HTML also has six levels of headings, which use the elements < h1 >, < h2 >, < h3 >, < h4 >, < h5 >, and < h6 >. While displaying any heading, the browser adds one line before and one line after that heading.

< h1 > Heading-1 < /h1 >
< h2 > Heading-2 < /h2 >
< h3 > Heading-3 < /h3 >
< h4 > Heading-4 < /h4 >
< h5 > Heading-5 < /h5 >
< h6 > Heading-6 < /h6 >
< h1 > headings should be used for main headings, followed by < h2 > headings, then the less important < h3 >, and so on.

The output after using this above tags are:-

Heading-1

Heading-2

Heading-3

Heading-4

Heading-5
Heading-6
